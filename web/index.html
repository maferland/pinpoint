How it works

A slash command opens your browser. You annotate. The agent picks up where it left off, now with structured data.

You take a screenshot Of whatever you're working on. A page in your dev server, a Storybook story, an iOS simulator, a Figma export.
Run /pinpoint-review screenshot.png The browser opens. You click to drop pins, drag to draw boxes, type comments. ⌘Enter saves each one.
Hit Send The agent gets structured JSON back. Coordinates as percentages (resolution-independent), comments as the source of truth.
The agent works each annotation as a discrete fix One task per pin. Round-trip is usually under a minute.

What Pinpoint sends back

{
  "annotations": [
    {
      "number": 1,
      "image": "/tmp/screenshot.png",
      "box": { "x": 10.2, "y": 5.3, "width": 35.0, "height": 12.5 },
      "comment": "Button text is truncated on mobile"
    }
  ]
}

Use it with your agent

Pinpoint speaks two protocols. Pick whichever fits where you already work.

Claude Code

Slash command, no setup beyond install.

/pinpoint-review /tmp/screenshot.png

Cursor, Aider, anything with MCP

Four tools: create_review, add_image, get_annotations, list_reviews. Same browser UI, same return shape.

claude mcp add pinpoint \
  -- bun ~/.pinpoint/src/main.ts --stdio

Direct CLI

Spawns the server, opens the browser, blocks until you hit Send, prints JSON to stdout. Pipe it anywhere.

pinpoint review screenshot.png \
  --context "Login page after auth changes"

Try the loop in 10 seconds

After install, no screenshot needed. Opens a sample session in your browser with starter pins.

pinpoint demo

Send a session to your designer

A review packages into a .pinpoint.zip. Hand it to anyone with Pinpoint installed. They add their pins, send it back, you re-import. Visual review without locking everyone into the same agent.

Export

Click the ⬇ button in the toolbar, or from the command line:

pinpoint export <reviewId>

Open someone else's bundle

Behaves like pinpoint review. Same browser UI, blocks until you hit Send, prints JSON.

pinpoint open bundle.pinpoint.zip
